# DSA in C++ from Scratch.
This is the repository I've made to track all the stuff I've learnt and applied.

The goal? Just keep going. Get the fundamentals down.

## For each structure: implement from scratch, then immediately do 3–5 targeted LeetCode problems on it.
  - Dynamic array → two-pointer / sliding-window
  - Stack → valid parentheses, min-stack, monotonic-stack
  - Queue → BFS problems
  - Linked list (singly + doubly) → reverse, cycle detection, merge two lists
  - BST → insert / search / validate, in-order traversal
  - Hash map → two-sum family, frequency / grouping
  - Min-heap → top-k, merge-k-lists, median of a stream
## Algorithms, also C++
Binary search, merge sort, quicksort — then one targeted LeetCode each for divide-and-conquer, greedy, and DP.
